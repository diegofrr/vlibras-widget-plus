# vlibras-widget-plus
Repositório de scripts com funcionalidades adicionais para o VLibras Widget.

🚧 Em desenvolvimento...

## Scripts
### **<a href="https://raw.githubusercontent.com/diegofrr/vlibras-widget-plus/main/translate-my-text.js">translate-my-text.js</a>**

**Descrição:**<br>
Adiciona ao Widget a possibilidade de receber para tradução textos passados pelo usuário.

**Como aplicar:**<br>
Após o Widget ser iniciado (a mostrar o avatar), abra seu _devtools_ e execute o _script_ dentro do console.
